# Admin Panel - Angular

Complete Angular admin panel for FashionNow E-Commerce.

## Setup

```bash
npm install
ng serve
```

Access at: http://localhost:4200

## Login Credentials

- Email: admin@fashionnow.com
- Password: admin123

## Features

- ✅ Add Products (with image upload)
- ✅ List Products
- ✅ Delete Products
- ✅ View Orders
- ✅ Update Order Status
- ✅ Admin Authentication

## Backend Required

Ensure backend is running on http://localhost:4000

## Complete Stack

- Backend: http://localhost:4000
- Frontend: http://localhost:4200 (customer)
- Admin: http://localhost:4200 (this admin panel)

Note: Run admin on different port if frontend is running:
```bash
ng serve --port 4201
```
